# Εγκατάσταση-ArchLinux
Την εγκατάσταση των ΑrchLinux αποφάσισα να την κάνω σε usb. Για να υλοπιοιθεί η εγκατάσταση χρειάστηκα δύο usb. Το πρώτο χρησιμοποιήθηκε ως ένα Arch live iso installation Usb και στο δευτέρο έγινε η εγκατάσταση των Αrch. 

Μπορείτε να δείτε το Νeofetch των ArchLinux στο αντίστοιχο [asciinema](https://asciinema.org/a/527109)
