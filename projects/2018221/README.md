## Εργασία 2: Get familiar with basic commands, reading documentation and editing files
Έγινε εγκατάσταση στο ubuntu του Vim editor, του file manager ranger καθώς και της εφαρμογής the fuck. Αρχικά είδαμε μερικές απο τις βασικές εντολες του linux καθώς και πως κάνουμε διόρθωση και επεξεργασία ενός αρχείου με την βοήθεια του editor vim. Στην συνέχεια περιηγηθήκαμε στους φακέλους και τα αρχεία του υπολογιστή μέσω του συστήματος διαχείρισης φακέλων ranger και τέλος είδαμε πως λειτουργέι η εφαρμογή the fuck.

* [x] Εξοικείωση με τον editor vim (copy,paste,undo,redo,search).
