# Παραδοτέο 1

Στο μάθημα Διάδραση Ανθρώπου-Υπολογιστή ευελπιστώ να καταλάβω και να χρησιμοποιήσω διάφορες τεχνολογίες που στοχεύουν στη διάδραση και την αποδοτικότητα στο τομέα του προγραμματισμού. Θέλω να μάθω τόσο για το τερματικό και τις λειτουργίες που μπορεί να προσφέρει όσο και για το γραφικό περιβάλλον. Θέλω να δώσω ιδιαίτερη ουσία στις βάσεις των σημερινών υπολογιστών καθώς και το ποιος τις διατύπωσε πρώτος και πως εξελίχθησαν στη συνέχεια. Μέσω του μαθήματος πιστεύω ότι μπορώ να μάθω τα παραπάνω αφού θα ασχοληθούμε με μια μεγάλη ποικιλία εργαλείων και τεχνολογιών όπου προσωπικά δεν έχω ξαναχρησιμοποιήσει ή δεν έχω χρησιμοποιήσει τόσο πολύ, όπως για παράδειγμα εργαλεία όπως το pandoc ,lua filters ,spacevim και άλλα.
