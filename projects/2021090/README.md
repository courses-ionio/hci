## Εβδομάδα Πρώτη - Εισαγωγή

<br /> 

&nbsp;&nbsp;&nbsp;Αντιλαμβάνομαι το μάθημα “επικοινωνία ανθρώπου υπολογιστή” ως μια εισαγωγή στον τρόπο με τον οποίο ο υπολογιστής διαχειρίζεται πληροφορίες και ποίος είναι ο ρόλος του ανθρώπου στην διαδικασία αυτή. Τα γραφικά και το animation είναι δύο πτυχές των υπολογιστών που με γοητεύον οι δυνατότητές τους. Θέλω επομένως να κατανοήσω ποίος είναι ο ρόλος τους στην διάδραση μαζί με την γενικότερη ιδέα αυτής. Θεωρώ πως μέσω του μαθήματος αυτό θα μπορέσω να καταλάβω καλύτερα που μπορούν να εφαρμοστούν και πώς μπορώ να ασχοληθώ μαζί τους στο μέλλον. Μέσα από τις εργασίες των διάφορων παραδοτέων θέλω να μάθω πως οργανώσω και να διαχειριστώ ένα πρότζεκτ .Ειδικότερα θέλω να μάθω scripting για αυτοματισμό και να πειραματιστώ με διάφορα εργαλεία ώστε να συγκεντρώσω διάφορα εφόδια στην διάθεσή μου. Με τις γνώσεις αυτές θα μπορώ να έχω καλύτερο έλεγχο στις καθημερινές μου δραστηριότητες μου στην καθημερινότητα και στο πανεπιστήμιο. Επίσης θα είμαι σε καλύτερη θέση να δημιουργήσω ή να συνεισφέρω σε κάποιο έργο αποκτώντας ανάλογη εμπειρία από δουλεύοντας σε ένα ομαδικό περιβάλλον που προσομοιώνει πραγματικές συνθήκες.

## Εβδομάδα Δωδέκατη - Τελική αναφορά

### Συμπεράσματα
&nbsp;&nbsp;&nbsp;Το μάθημα επικοινωνία ανθρώπου υπολογιστή έφτασε στο τέλος του, παρά τις όποιες δυσκολίες προέκυψαν έπειτα από εντατική προσπάθεια όλα τα παραδοτέα ολοκληρώθηκαν. Η εμπειρία μου αυτές τις εβδομάδες χαρακτηρίζεται κυρίως από τον τρόπο σκέψης που τα βέντεο-κουιζ και οι εργασίες ανάγκασαν να αναπτύξω. Αρχικά ακούγοντας πρωτοπόρους(τους οποίους δεν γνώριζα ) του κλάδου να μιλούν οι ίδιοι για τις εμπειρίες και τις απόψεις τους με έκανε να αναρωτηθώ και να αμφισβητήσω πολλές από τις εφαρμογές διάδρασεις που χρησιμοποιώ και εγώ ο ίδιος στην καθημερινότητα μου για τον τρόπο που λειτουργούν. Τέτοια θέματα ήταν η χρήση του ποντικιού, η “δυσκολία του πληκτρολογίου” καθώς οι “έξυπνες” εφαρμογές που παίρνουν τον έλεγχο από τον χρήστη και αντί να έχουν ακριβή ανταπόκριση  προσπαθούν να μαντέψουν ποια θα είναι η επόμενη κίνηση του πέφτοντας έξω. Κλονίστηκαν επίσης κάποιες πεποιθήσεις που είχα για τον χώρο ,ως πιο ‘’αγαθό’’ σε σχέση με άλλους ,και αντιλήφθηκα ότι μια ευρεσιτεχνία κυριαρχεί όχι πάντα λόγω ιδιοφυούς σχεδιασμού αλλά επειδή υπάρχουν κερδοσκοπικά συμφέροντα που θα κάνουν την ανάλογη προώθηση . Πράγματι το συμφέρον αυτή την εποχή(και όχι μόνο) συστήνει την εξέλιξη του υπολογιστή και παραγκωνίζει τον άνθρωπο έτσι πλέον είμαι πιο κριτικός απέναντι σε ζητήματα που προβλημάτισαν στο παρελθόν ,όπως η τεχνητή νοημοσύνη η οποια για μπορεί να λειτουργήσει και να παρέχει εύκολα δεδομένα παραβιάζει τα προσωπικά δεδομένα του καθένας. Έχοντας πλέον ένα πιο σταθερό σημείο εκφρασης που ανταποκρίνεται περισσότερο στην πραγματικότητα παρά στις ταινίες επιστημονικής φαντασίας. Αν και όλα αυτά ίσως προαναγγέλλουν ένα ζοφερό μέλλον προσωπικά με αφορούν μέχρι ένα σημείο ως άνθρωπο και μελλοντικό επιστήμονα πληροφορικής; αυτό που έχει πραγματική σημασία για την ώρα είναι ότι σε εφαρμογές που χρησιμοποιώ καθημερινά για τις πανεπιστημιακές και μη υποχρεώσεις μου τα αντιμετωπίζω με περισσότερη κριτική σκέψη και προτιμώ να αφιερώσω περισσότερο χρόνο στο να μάθω να χειρίζομαι κάποια που θα μου επιτρέπει να είμαι πιο αποτελεσματικός. Έχοντας έρθει σε επαφή με διάφορα εναλλακτικά εργαλεία για την περάτωση των εργασιών καθώς και μπαίνοντας σε μια διαδικασία αναζήτησης για την αντιμετώπιση των προβλημάτων που εμφανίζονταν στην πορεία αυτό είναι πλέον πολύ πιο εύκολο. Θωρώ λοιπόν πως έχω χτίσει τις απαραίτητες βάσεις για να ασχοληθώ με μεγαλύτερη ακρίβεια και αποτελεσματικότητα αλλά και οργάνωση σε όποιο πρότζεκτ με ενδιαφέρει όπως να ασχοληθώ πιο εντατικά με τα γραφικά και συγκεκριμένα το blender και την δημιουργία δικών script ή εργαλείων ή ακόμα να συνεισφέρω ενεργά στα github της κοινότητας του. Κάτι τελευταίο που αφορά τα γραφικά είναι επιβεβαίωση ότι η υψηλή ανάλυση δεν είναι απαραίτητη όταν θες να περάσεις ένα μήνυμα ,έτσι στην διάδρση σημασία έχει να τονιστούν τα χαρακτηριστικά εκείνα που καθιστούν το μήνυμα σαφές για μια προσπάθεια επικοινωνίας
