## ΕΙΣΑΓΩΓΗ-Οι Προσδοκίες μου από το Μάθημα και τα Πρότζεκτ

Από το συγκεκριμένο μάθημα περιμένω να διδαχθώ έναν διαφορετικό τρόπο σκέψης, από αυτόν που είχα τόσα χρόνια, σχετικά με τον τρόπο που χρησιμοποιώ και χειρίζομαι τον υπολογιστή. Με την εξοικείωση μου στο σύστημα του arch ευελπιστώ να πετύχω αυτό τον στόχο μου. Επίσης, πολύ σημαντικό για μένα είναι η εξοικείωση μου με το git και το github, αλλά και η βελτίωση μου στη συνεργατικότητα μέσω του οργανισμού. Πολύ σημαντικό μέρος του μαθήματος για μένα είναι και τα βιντεάκια στο ed puzzle όπου μέσα από αυτά θα μάθω περισσότερα πράγματα σχετικά με την ιστορία της διάδρασης αλλά και με τις τόσες ιδέες που, παρόλο που μπορεί να ήταν καλύτερες και πιο ευνοικές για τον άνθρωπο, τελικά δεν επικράτησαν αλλά στη θέση τους επικράτησαν άλλες λόγω της ευχρηστίας τους ή λόγω marketing. Τέλος, ελπίζω μέσω του μαθήματος να μπορέσω να ανακαλύψω περισσότερα πράγματα σχετικά με τα ενδιαφέροντά μου, που είναι η υλοποίηση βιντεοπαιχνιδιών αυτών καθ' αυτών αλλά και οι διάφορες συσκευές, τεχνολογίες και καινοτομίες που μπορούν να δώσουν στον παίκτη τη καλύτερη δυνατή εμπειρία και αλληλεπίδραση με τα βιντεοπαιχνίδια.

## Παραδοτέο 2o- Άσκηση γραμμής εντολών (Linux install)

Για την υλοποίηση του 2ου παραδοτέου έκανα τα arch Linux installation με την βοήθεια του archwiki και έπειτα με την χρήση του asciinema βιντεοσκόπησα τις αντίστοιχες εντολές neofetch και journalctl -b όπως μπορείτε να δείτε κάτω στα links. Πιο επεξηγηματικά  έκανα dual boot τα arch Linux με τα windows απλός η δυσκολία που συνάντησα είναι πως στον υπολογιστή μου από ότι φαίνεται τα windows δεν είχανε δημιουργήσει EFI partition οπότε χρειάστηκε να κάνω format στον υπολογιστή μου. Έπειτα ξανακατεβασα και εγκατέστησα τα windows όπου αυτή την φορά δημιουργήθηκε κανονικά το EFI partition και μπόρεσα να κάνω κανονικά το installation του arch Linux όπου και δούλεψε.

- [Neofetch](https://asciinema.org/a/534340)
- [Journalctl](https://asciinema.org/a/534339)
